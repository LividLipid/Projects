# Projects
Practice projects

Customizable menu system written in August 2015.
Functionality includes reading RSS feeds and a to-do list.
The Purpose was to learn several different techniques, including:

Unit testing and TDD:
- The entire project was written tests first.

Design Patterns:
- Composite.
- Iterator.
- Special Case.
- Factory Method.

Algorithms:
- Tree traversal.

Arcitechture:
- Decoupling of application architecture from UI+persistence.
- Composition vs inheritance.
- - Dependency inversion.

Clean Code:
- Single responsibility classes.
- Single purpose functions.
- Naming style.
- No repetition.
- Law of Demeter.
